# Dossier templates

## Raison d'être
Le dossier `templates` contient tous les gabarits HTML utilisés par l'application. Ces gabarits permettent de créer et de structurer le contenu des pages web.

## Fichiers principaux
- `index.html`:  Gabarit de la page principale, c'est-à-dire l'interface de la calculatrice.

## Dépendances / Hypothèses
- Communique avec `app.py` via des requêtes HTTP.
- A accès au dossier `static` pour tout fichier css ou autre ressource statique nécessaire.
