# Dossier Tests

## Raison d'être
Le Dossier `tests` contient tous les tests unitaires pour l’application web. Il permet de vérifier le fonctionnement attendu des fonctions de calcul arithmétique et détecter tout comportement inattendu des fonctions testées.

## Fichiers principaux
- `test_operators.py`: tests qui couvrent les fonctions arithmétiques `add`, `subtract`, `multiply`, `divide`.
- `test_app.py`: tests qui couvrent la fonction `calculate()` et la gestion des expressions invalides.
