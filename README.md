# LOG3000 - Calculatrice simple
Équipe 15

## Objectif
L'objectif de ce projet est de documenter et collaborer dans le developpement d'une application web de calculatrice.

# Prérequis
Avant de lancer le projet, il faut avoir d'installé localement:
- Python 3.10+
- pip
- Git
Ainsi qu'avoir un compte Github.


## Instructions d'installation
1. Cloner le dépôt localement

2. Installer les dépendances
`pip install flask`

3. Lancer l'application
`python app.py`

L'application sera disponible à l'addresse `http://127.0.0.1:5000/`
