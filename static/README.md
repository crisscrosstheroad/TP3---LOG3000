# Dossier static

## Raison d'être
Le dossier `static` contient tous les atouts statiques, c'est-à-dire tout fichier css, images ou autre contenu statique, nécessaires à l'application web.

## Fichiers principaux
- `style.css`:  Contient tout le style css pour définir l'apparence de la calculatrice.

## Dépendances / Hypothèses
- Les fichiers dans le dossier `templates` utilisent les atouts statiques contenus dans ce dossier.
